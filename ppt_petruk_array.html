SLIDE 02 / 10
TAKTIK-01

Apa itu Array?

Bayangkan kamu punya 5 nilai ujian. Tanpa array, kamu harus buat 5 variabel terpisah (a, b, c, d, e). Kalau datanya 100? Bikin 100 variabel? Tentu tidak efisien.

Array = satu variabel yang bisa menyimpan banyak data sejenis, berurutan di memori, dan setiap data punya nomor urut (indeks).

3 Sifat Array:

  • Semua elemen harus tipe data sama (misal: semua int)
  • Ukurannya tetap sejak dideklarasikan, tidak bisa berubah
  • Indeks dimulai dari 0, bukan 1

SLIDE 03 / 10
TAKTIK-02

Cara Menulis Array

Rumus dasarnya: tentukan tipe data, nama array, dan jumlah elemen yang mau ditampung:

RUMUS SINTAKS
tipe_data namaArray[jumlah_elemen];

① Deklarasi kosong (isi nanti)

MAIN.CPP
int nilai[5];
// Menyiapkan 5 tempat untuk angka
// Belum ada isinya (garbage value)

Ibarat menyiapkan 5 laci kosong — nanti baru diisi satu per satu lewat kode atau input user.

② Langsung diisi saat dibuat

MAIN.CPP
int n[5] = {14, 65, 78, 54, 90};
// 5 laci langsung terisi

Lebih aman karena kamu tahu persis isinya. Tidak ada garbage value.

Ingat!  Ukuran array harus angka tetap (konstanta). Kamu tidak bisa menulis int arr[x] jika x adalah variabel biasa.

SLIDE 04 / 10
TAKTIK-03

Kenapa harus Array?

Coba bandingkan: kamu mau simpan 5 nilai input dari user.

✗ TANPA ARRAY
MASALAH.CPP
int a;  // cuma 1 variabel

for(int i = 0; i < 5; i++) {
    cout << "Input: ";
    cin >> a;
    // data LAMA langsung hilang!
}
cout << a;
// hanya tampil nilai TERAKHIR

⚠ Masalah: Variabel a cuma punya 1 tempat. Setiap cin baru, data lama tertimpa dan hilang!

✓ DENGAN ARRAY
SOLUSI.CPP
int a[5];  // 5 tempat tersedia

for(int i = 0; i < 5; i++) {
    cout << "Input: ";
    cin >> a[i];
    // masuk ke kotak ke-i
}
for(int i = 0; i < 5; i++)
    cout << a[i] << " ";
// semua 5 data tampil!

✓ Solusi: Setiap data punya tempat sendiri (indeks 0-4). Tidak ada yang tertimpa.

SLIDE 05 / 10
TAKTIK-04

Contoh: Isi Langsung

Nilai array ditulis langsung di kode saat deklarasi. Lalu kita tampilkan isinya pakai loop for:

MAIN.CPP
#include <iostream>
using namespace std;

int main() {
    // Deklarasi + langsung isi 5 nilai
    int n[5] = {14, 65, 78, 54, 90};

    // Cetak semua elemen pakai loop
    for(int j = 0; j < 5; j++)
        cout << "Nilai ke-" << j
             << " = " << n[j] << endl;

    return 0;
}

▶ Hasil di terminal:

OUTPUT
Nilai ke-0 = 14
Nilai ke-1 = 65
Nilai ke-2 = 78
Nilai ke-3 = 54
Nilai ke-4 = 90

Perhatikan!  Indeks dimulai dari 0, bukan 1. Elemen terakhir = n[4], bukan n[5].

SLIDE 06 / 10
TAKTIK-05

Contoh: Input dari User

Kali ini array-nya kosong dulu. User yang mengisi nilainya saat program berjalan lewat cin:

MAIN.CPP
#include <iostream>
using namespace std;

int main() {
    int a[7];  // siapkan 7 tempat kosong

    // Loop 1: User isi satu per satu
    for(int i = 0; i < 7; i++) {
        cout << "A[" << i << "] = ";
        cin >> a[i];  // masuk ke kotak ke-i
    }

    // Loop 2: Tampilkan semua isi array
    cout << "\nIsi array:\n";
    for(int i = 0; i < 7; i++)
        cout << "A[" << i << "] = " << a[i] << endl;

    return 0;
}

Perbedaan!  Di slide sebelumnya (Taktik-04), nilai ditulis langsung di kode. Di sini, nilai diisi oleh user saat program jalan lewat cin >> a[i]. Dalam materi, cara ini disebut "pengisian dinamis".

TAKTIK-06

Akses & Ubah Elemen

Setiap elemen punya alamat indeks. Kamu bisa baca atau ganti nilainya:

MAIN.CPP
int arr[5] = {10, 20, 30, 40, 50};

// BACA elemen
cout << arr[0];  // cetak 10
cout << arr[4];  // cetak 50

// UBAH elemen indeks ke-2
arr[2] = 99;
cout << arr[2];  // sekarang 99

⚠ Awas Out-of-Bounds! Akses indeks di luar batas (misal arr[10] padahal cuma ada 5 elemen) bisa bikin program crash atau hasilnya kacau. C++ tidak akan peringatkan kamu!

SLIDE 08 / 10
DEBRIEFING

Rangkuman Array

Semua taktik yang sudah kita bahas, dipadatkan jadi 5 poin visual.

RANGKUMAN-01

Penulisan Array Ada 2 Cara

① Deklarasi kosong

int data[5];

Siapkan tempat dulu, isi belakangan.

② Langsung diisi

int data[5]={10,30,50..};

Nilai langsung ditulis saat deklarasi.

RANGKUMAN-02

Indeks Mulai dari 0

[0]
[1]
[2]
[3]
[4]

5 elemen → indeks 0 s.d. 4, bukan 1 s.d. 5.

RANGKUMAN-03

Isi & Baca Elemen

data[3] = 100;

Pakai operator assignment seperti variabel biasa — tinggal tambahkan indeksnya.

RANGKUMAN-04

Looping = Kekuatan Utama Array

for (int i = 0; i < 5; i++) { cout << data[i] << endl; }

Satu perulangan cukup untuk mengakses semua elemen, tak perlu ditulis satu-satu.

RANGKUMAN-05

Array Bisa Lebih dari 1 Dimensi

int data[3][4];

Larik 2D = "tabel" dengan baris & kolom. Total elemen = 3 × 4 = 12, dari data[0][0] sampai data[2][3].

Sel merah = data[1][2] — baris ke-1, kolom ke-2.
