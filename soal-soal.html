SOAL 01 / 05

Apa keluaran dari potongan program berikut?

MAIN.CPP
int data[5] = {10, 20, 30, 40, 50};
int total = 0;

for (int i = 1; i < 4; i++) {
    total += data[i];
}

cout << total << endl;
Jawaban: B (90). Loop mulai dari i = 1 sampai i < 4, jadi indeks yang dijumlahkan adalah 1, 2, 3 → data[1]+data[2]+data[3] = 20+30+40 = 90. Indeks 0 dan 4 tidak ikut terhitung.
SOAL 02 / 05

Baris mana pada kode berikut yang menyebabkan error saat kompilasi?

MAIN.CPP
int n = 5;
int nilai[n];        // baris A
int skor[5];       // baris B
skor[0] = 100;      // baris C
skor[5] = 90;       // baris D
Jawaban: A. Ukuran array pada C++ standar harus konstanta, bukan variabel biasa seperti n, sehingga baris A tidak valid di banyak compiler standar. Baris D (skor[5]) sebenarnya juga bermasalah — itu out-of-bounds karena indeks maksimal skor adalah 4 — tapi itu tidak menyebabkan error kompilasi, hanya perilaku tak terduga saat program berjalan.
SOAL 03 / 05

Diberikan larik 2 dimensi berikut, apa nilai dari matrix[1][2]?

MAIN.CPP
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

cout << matrix[1][2] << endl;
Jawaban: C (6). Indeks baris 1 berarti baris kedua → {4, 5, 6}. Indeks kolom 2 berarti elemen ketiga di baris itu → nilainya 6.
SOAL 04 / 05

Apa keluaran dari potongan program berikut?

MAIN.CPP
int angka[6] = {5, 12, 8, 3, 19, 7};
int terbesar = angka[0];

for (int i = 1; i < 6; i++) {
    if (angka[i] > terbesar) {
        terbesar = angka[i];
    }
}

cout << terbesar << endl;
Jawaban: B (19). Kode ini mencari nilai terbesar dalam larik dengan membandingkan tiap elemen ke terbesar. Nilai tertinggi dalam larik tersebut adalah 19.
SOAL 05 / 05

Manakah pernyataan yang benar tentang array di C++?

Jawaban: C. Array bersifat homogen — semua elemennya harus tipe data yang sama. Ukurannya tetap sejak dideklarasikan (bukan A), indeks dimulai dari 0 (bukan B), dan C++ tidak memperingatkan akses out-of-bounds (bukan D) — itu tanggung jawab programmer.
